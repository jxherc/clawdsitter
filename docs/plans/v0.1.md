# v0.1 plan — Foundations (config + log)

## Tasks

### v0.1-config — `src/config.js`
- `DEFAULTS` object (claudeCommand, resumePhrase, backoffSteps, resetGraceSeconds,
  fallbackWaitMinutes, maxResumes, injectDelayMs, patterns{transient,limit}, notify{toast}).
- `parseDuration(v)` → ms. Suffixes `ms|s|m|h`; bare number = ms; junk throws.
- `loadConfig({home}?)` → deep-merge user `config.json` over DEFAULTS. `home` defaults to
  `CLAWDSITTER_HOME` env or `~/.clawdsitter`. Missing/!valid file → DEFAULTS.
- `configHome()` helper.

Tests (>=8): DEFAULTS shape; parseDuration s/m/h/ms/number; junk throws; no-file→DEFAULTS;
partial merge; nested deep-merge (notify/patterns); home override via arg + env.

### v0.1-log — `src/log.js`
- `createLog(logPath, {maxLines=500})` → `{ append(event, msg), path }`.
- append writes `YYYY-MM-DD HH:mm:ss  <event>  <msg>` (newlines in msg collapsed to spaces),
  creates parent dir, trims file to last `maxLines`.
- `defaultLogPath(home)` helper.

Tests (>=8): writes a line; format/timestamp; multi-append accumulates; creates missing dir;
trim to default 500; trim to custom maxLines; newline sanitize; empty msg ok; path exposed.
