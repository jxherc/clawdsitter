# v0.5 plan — PTY host + notifications

## Task v0.5-ptyhost — `src/ptyHost.js`
`createPtyHost({ command, args, cwd, env, cols, rows, onData, stdin, stdout, spawn })`
→ `{ write, resize, kill, onExit, restore, pid }`.
- default `spawn` = `@lydell/node-pty`.spawn; defaults stdin/stdout = process streams.
- child.onData → `stdout.write(d)` + `onData(d)`.
- stdin 'data' → `child.write(...)`; raw-mode TTY stdin, restore on exit.
- stdout 'resize' → `child.resize(cols,rows)`.
- child.onExit → restore() then fire registered exit callbacks.

Tests (>=8): spawn gets command/args + cols/rows; output → stdout & onData; write→child.write;
resize→child.resize; onExit fires + restore removes stdin listener; stdin forwarded; raw-mode set
& restored; pid exposed; (real PTY) passthrough captures output; (real PTY) write reaches child.

## Task v0.5-notify — `src/notify.js`
`createNotifier(config, lib = node-notifier)` → `{ toast(title,msg), enabled }`.
- gated by `config.notify.toast !== false`; swallow lib errors; return whether it fired.

Tests (>=4): enabled by default calls lib with title/message; disabled doesn't call lib;
returns true/false accordingly; swallows a throwing lib.
