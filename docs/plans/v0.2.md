# v0.2 plan — Detector

## Task v0.2-detector — `src/detector.js`
- `stripAnsi(s)` — remove ANSI escape / control sequences.
- `classify(text, patterns = DEFAULTS.patterns)` → `{ kind: 'transient'|'limit'|null, match? }`.
  Strips ANSI, tests `limit` first (precedence), then `transient`. Patterns are regex *sources*
  (strings) compiled case-insensitive.
- `createWatcher({ patterns, carryover = 256 })` → `{ feed(chunk), tail }`. Prepends the last
  `carryover` chars to each new chunk (boundary-safe), classifies that window, returns the
  detection or `null`. (Episode/idempotency is the supervisor's job, not the watcher's.)

## Tests (>=8)
real transient (401), real cert error, real limit string, "hit your usage limit", healthy→null,
limit precedence over transient, stripAnsi + classify on ANSI-wrapped error, custom pattern
override, watcher detects across a chunk boundary, watcher returns null on healthy chunk.
